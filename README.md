# recruitment

## 功能介绍
### 管理员 admin

1. 审核HR注册信息 
2. 审核求职者注册信息
3. 审核招聘信息

### 招聘者 recruiter

1. 发布招聘信息
2. 接收求职者简历
3. 筛选查看求职者，向求职者发送邀约

### 求职者 candidate

1. 上传简历附件/填写在线简历
2. 投递简历
3. 浏览筛选职位列表
4. 分享职位
5. 投递记录，进度

### 聊天功能

   1.求职者与HR交流

### 登录注册功能   

​	springsecurity + jwt

## 数据库设计

### 管理员 admin
用户id、用户名、密码

### 求职者 candidate
用户id、用户名、密码、手机号、性别、出生年月、所在城市、简历附件

### 招聘者 recruiter
用户id、真实姓名、性别、担任职务、企业名称、企业所在地、企业类型

### 职位信息 position
职位名称、职位描述、职位亮点（可选）、职位类别、学历/经验要求、职位关键词、工作地址、薪资范围
